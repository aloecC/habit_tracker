# 🌿 EverBusy Habit Tracker Bot

EverBusy Habit Tracker — это продвинутый Telegram-бот на базе Django, созданный для формирования и отслеживания привычек. Бот использует методику «атомных привычек», помогая пользователям связывать полезные действия с приятными вознаграждениями или приятными привычками.

 🚀 Основные возможности

•  Гибкое управление действиями: Создание "Любимых действий" (приятных) и "Нужных действий" (полезных).
•  Локации и Награды: Привязка привычек к конкретным местам и системе поощрений.
•  Два типа привычек:
  •  ✨ Приятные привычки: Простые действия для удовольствия.
  •  📝 Полезные привычки: Сложные привычки с расписанием, локацией и связкой (награда или приятная привычка).
•  Умные уведомления: Автоматические напоминания в Telegram через Celery и Redis.
•  Публичность: Возможность делать свои привычки публичными для сообщества.
•  Админ-панель: Управление всеми данными через стандартный интерфейс Django.

 🛠 Технологический стек

•  Backend: Python 3.13+, Django 5.x (ORM, Management Commands)
•  Бот: python-telegram-bot (https://github.com/python-telegram-bot/python-telegram-bot) (асинхронный)
•  База данных: PostgreSQL (рекомендуется) или SQLite
•  Фоновые задачи: Celery
•  Брокер сообщений / Кэш: Redis
•  Управление зависимостями: Poetry / Pip

 📋 Требования

•  Python 3.13 или выше
•  Redis сервер (запущенный локально или в Docker)
•  Токен Telegram бота (полученный у (@BotFather (https://t.me/BotFather))

 ⚙️ Установка и настройка

1. Клонируйте репозиторий
2. Установите зависимости
3. Настройте переменные окружения:
  Создайте файл .env в корне проекта или пропишите в settings.py
    TELEGRAM_BOT_TOKEN=your_token_here
    SECRET_KEY=django_secret_key
    DEBUG=True
    CELERY_BROKER_URL=redis://localhost:6379/0
4. Примените миграции
5. Создайте суперпользователя (для доступа к админке)


🏃 Запуск проекта

Для полной работы системы необходимо запустить 4 процесса в разных терминалах:

1. Redis (redis-server)
2. Celery Worker(celery -A config worker -l INFO -P eventlet)
3. Celery Beat(celery -A config beat -l info --scheduler django_celery_beat.schedulers:DatabaseScheduler)
4. Telegram Bot(python manage.py runbot)


📐 Архитектура базы данных

•  User: Расширенная модель пользователя с telegram_chat_id.
•  Action (LikeAction / NeedAction): Справочник действий.
•  Location: Места выполнения привычек.
•  Reward: Возможные поощрения.
•  HabitNice: Приятные привычки (Действие + Локация).
•  HabitUseful: Полезные привычки (Действие + Локация + Время + Периодичность + Связка с Reward или HabitNice).

---

 📝 Использование бота

1. Найдите бота в Telegram и нажмите /start.
2. Используйте меню для добавления базовых сущностей (Локации, Действия).
3. Создайте Приятную привычку.
4. Создайте Полезную привычку, привязав её к созданной ранее приятной привычке или награде.
5. Бот автоматически пришлет вам сообщение, когда придет время выполнять привычку.
